> 我们拥有如下代码，对其中的一段代码开启了`VMProtect`中的`虚拟化`功能

```C++
#include <stdio.h>
#include <Windows.h>
#include "VMProtectSDK.h"
int main() {
	int a = 0, b = 0, c = 0;
	while (true) {
		VMProtectBegin("VMProtectBegin");
		a += 10;
		b += 100;
		c = a + b;
		VMProtectEnd();
		printf("%d %d %d\n", a, b, c);
		Sleep(1000);
	}
	return 0;
}
```
